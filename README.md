## 🚀 About the Portfolio
This website is a custom-built, fully responsive, and interactive digital resume. Designed with a clean, dark-mode "hacker" aesthetic, it features custom parallax scrolling, floating particles, and glassmorphism elements to provide a premium user experience.

### Key Features:
- **Custom Animations**: Figma-style parallax, magnetic buttons, and 3D card tilt effects.
- **Dynamic Content**: Auto-typing text, animated skill bars, and a responsive navigation system.
- **Responsive Design**: Flawless experience across desktop, tablet, and mobile viewing.
- **Interactive Forms**: Built-in contact form layout ready for backend integration.

## 💻 Tech Stack
- **Structure**: Semantic HTML5
- **Styling**: Modern Vanilla CSS3 (Custom Variables, Flexbox, Grid, Glassmorphism)
- **Logic & Animation**: Vanilla JavaScript (Intersection Observers, Canvas API, RequestAnimationFrame)
- **Icons**: FontAwesome

## 📂 Featured Projects Showcased
1. **AI RESUMANY**: An intelligent resume analysis system leveraging NLP and ML for ATS compatibility and skill gap analysis.
2. **Real-Time Collaborative Coding Platform**: A web-based synchronized coding environment with WebSocket integration.
3. **SecMan (Security Manager)**: A comprehensive vulnerability scanner for detecting SQL injection, XSS, and misconfigurations.
